AI Assistant
help me find large files
I can help with that. Here's a command to find the largest files in the current directory: 
find . -type f -size +10M | sort -rh | head -n 5
This will show the 5 largest files over 10MB.
find TODOs in my code
To find all TODOs in your codebase, you can use grep: 
grep -r "TODO" .
This will recursively search for "TODO" in all files in the current directory.
